
嗅嗅浏览器 一键提取网页图片应用

这是本人一下午写的APP，所以功能不是非常完善请见谅。
感觉可能让“宅男”看图能更方便一些，所以随手就扔到了App Store上。
崇尚着开源精神，所以代码公开，主要使用Swift语言，SDWebViewImage和HTMLReader框架，实现了图片提取、相册浏览的功能。
